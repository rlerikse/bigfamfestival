## How to Add Specifications

1. Install Spec-Kit in your repository:
   ```bash
   curl -fsSL https://raw.githubusercontent.com/rlerikse/es-spec-kit-context/sync/context/speckit.sh | bash
   ```

2. Create a specification:
   ```
   /speckit.quickstart "Your feature description"
   ```

3. Push to trigger sync:
   ```bash
   git push origin main
   ```

4. Specs will be aggregated here automatically.
